# Dashboard Bibliográfico — UNSAM

Exploración interactiva de la bibliografía de las carreras de Sociología y Antropología de la Universidad Nacional de San Martín (UNSAM).

## Funcionalidades

- **Dashboard**: 6 gráficos interactivos (épocas, género, región, país de instituciones, top instituciones, disciplinas) con filtros por carrera, año del plan y materias.
- **Comparar**: Vista lado a lado para comparar bibliografía entre carreras o materias específicas.

## Ejecución local

```bash
pip install -r requirements.txt
python app.py
```
